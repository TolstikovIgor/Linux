1. Навигация по файловой системе
Перечислить, какие параметры команды cd позволят быстро вернуться в домашний каталог
быстро вернуться в домашний каталог: cd или cd ~

позволят перейти на уровень выше
cd ../

2. Управление файлами и каталогами и текстовые редакторы.
Создать файл с наполнением, используя несколько способов
echo "Это строка" > h
echo "Это текстовая строка" > файл.txt
echo "Это вторая текстовая строка" >> файл.txt

Использовать разобранные текстовые редакторы для наполнения файлов данными
vi файл.txt
vim myfile
nano myfile

Создать копии созданных файлов
cp (copy): cp file1 file2

создать несколько каталогов с подкаталогами
mkdir -p /data/rs1 /data/rs2 /data/rs3
md data\rs1 data\rs2 data\rs3

перенести несколько файлов в созданные каталоги
mv [OPTIONS] SOURCE DESTINATION
чтобы переместить файл file1 из текущего рабочего каталога в каталог /tmp
mv file1 /tmp
переименовать файл
mv file1 file2
перемещения каталого
mv dir1 dir2
Чтобы переместить несколько файлов и каталогов
mv file1 file2 dir1

3. * Используя дополнительный материал, настроить авторизацию по SSH с использованием ключей.
mkdir ~/.ssh/
cat id_rsa.pub > ~/.ssh/authorized_keys
chmod 700 ~/.ssh/
chmod 600 ~/.ssh/authorized_keys
ssh-copy-id
